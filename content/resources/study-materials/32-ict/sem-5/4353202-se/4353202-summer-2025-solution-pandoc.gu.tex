સ્ટ્રક્ચરનું જ્ઞાન વિના ઇનપુટ અને અપેક્ષિત આઉટપુટ ટેસ્ટ કરીને કાર્યાત્મક આવશ્યકતાઓને માન્ય

   18{-65 વર્ષ       0   0{-}17   66{-}120   120}
      |                |     |      |       |
      v                v     v      v       v
   [માન્ય]         [અમાન્ય ઇનપુટ્સ]
\end{verbatim}

\textbf{ઉદાહરણ}: જોબ એપ્લિકેશન માટે વય વેલિડેશન

\item

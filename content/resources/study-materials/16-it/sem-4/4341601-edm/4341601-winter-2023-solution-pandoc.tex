A/B testing compares two versions to determine which performs better.

    Version A     Version B
    +{-{-}{-}{-}{-}{-}{-}+     +{-}{-}{-}{-}{-}{-}{-}+}
    | Red   |     | Blue  |
    | Button|     | Button|
    +{-{-}{-}{-}{-}{-}{-}+     +{-}{-}{-}{-}{-}{-}{-}+}
         |             |
         v             v
    50\% Traffic   50\% Traffic
         |             |
         v             v
    Results A     Results B
    +{-{-}{-}{-}{-}{-}{-}+     +{-}{-}{-}{-}{-}{-}{-}+}
    | 5\%    |     | 8\%    |
    | Click |     | Click |
    +{-{-}{-}{-}{-}{-}{-}+     +{-}{-}{-}{-}{-}{-}{-}+}
                     \^{}
                   Winner
\end{verbatim}
